Gobot
======

I am your pet, baby! I can do anything for you, just tell me...

## Installation

```sh
go get github.com/vishaltelangre/gobot
go install githhub.com/vishaltelangre/gobot
which gobot
```

## Sneak Peek Usage!

start `gobot` and type `help` in it.
